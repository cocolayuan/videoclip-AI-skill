# 智能划重点

> 为字幕添加智能关键词高亮，让重点一目了然

## 快速使用

```
用户: 帮这个视频添加智能划重点
用户: 字幕加高亮
用户: 智能标记关键词
```

## 功能特点

### ✨ 核心优势

1. **尊重中文语境** - 识别固定搭配，不拆分词语
2. **精准控制** - 每行最多2个高亮，避免过度标记
3. **智能优先级** - 强调词 > 核心名词 > 形容词 > 动词
4. **自动纠错** - 发现并修复转录错误

### 🎨 视觉效果

| 元素 | 样式 |
|------|------|
| 默认文字 | 白色、24pt、黑色描边 |
| 高亮关键词 | **黄色、28pt、加粗、黑色描边** |

## 工作流程

```
1. 读取字幕文件（SRT格式）
    ↓
2. 中文分词（jieba + 固定搭配识别）
    ↓
3. 关键词评分（考虑类别、上下文、词性）
    ↓
4. 每行选择 Top 2 关键词
    ↓
5. 生成 ASS 字幕（带内联样式标记）
    ↓
6. FFmpeg 烧录到视频
    ↓
【输出】带智能高亮的视频
```

## 核心算法

### 固定搭配识别

```python
FIXED_COLLOCATIONS = {
    '大家', '很多', '不会', '能建起来',
    '有一种', '群山环绕', '无论是',
    # ... 更多固定搭配
}
```

这些词组**必须整体高亮或不高亮**，绝不拆分。

### 关键词分类

| 类别 | 优先级 | 示例 |
|------|--------|------|
| 强调词 | 4 | 非常、十分、特别、最 |
| 核心名词 | 3 | 高铁、座位、空间、质量 |
| 正面形容词 | 3 | 宽敞、有利、伟大、细心 |
| 固定搭配 | 2 | 能建起来、群山环绕 |
| 能愿动词 | 1 | 可以 |

### 禁用词过滤

```python
STOP_WORDS = {
    '的', '了', '呢', '啊',  # 语气词
    '这', '那', '我', '你',  # 代词
    '然后', '或者', '不用',  # 连接词
    # ... 更多功能词
}
```

## 技术细节

### ASS 字幕内联样式

```ass
普通文字{\fs28\c&H00FFFF&\b1}高亮词{\r}继续普通文字
```

- `\fs28` - 字号28pt（+4）
- `\c&H00FFFF&` - 黄色（BGR格式）
- `\b1` - 加粗
- `{\r}` - 重置样式

### 字体大小兼容性

⚠️ **关键**：ASS文件**不要**设置 `PlayResX/PlayResY`，让字幕根据视频分辨率自动缩放。

```ass
[Script Info]
Title: 智能划重点字幕
ScriptType: v4.00+
WrapStyle: 0
ScaledBorderAndShadow: yes
# ❌ 不要添加 PlayResX 和 PlayResY
```

## 常见问题

### Q1: 为什么有些词没有高亮？

可能是：
1. 该词在禁用词列表中
2. 同行有更高优先级的词（限制≤2个）
3. 单字词默认不高亮

### Q2: 能自定义高亮颜色吗？

可以修改 `smart_highlight_v2.py` 中的颜色代码：
```python
# 黄色 &H00FFFF&  (BGR: FF FF 00)
# 红色 &H0000FF&  (BGR: 00 00 FF)
# 绿色 &H00FF00&  (BGR: 00 FF 00)
```

### Q3: 如何添加新的固定搭配？

编辑 `FIXED_COLLOCATIONS` 集合：
```python
FIXED_COLLOCATIONS = {
    '大家', '很多',
    '你的新词组',  # 添加在这里
}
```
